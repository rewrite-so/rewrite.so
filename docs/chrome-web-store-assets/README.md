# Chrome Web Store image assets

Final image assets for the Chrome Web Store listing.

Core message: **Write freely. Send confidently.**

The `imagegen/` PNG files are the current recommended upload set. Earlier deterministic screenshot and promo drafts have been removed to avoid uploading the wrong versions.

## Recommended upload set

- `store-icon-128.png` - 128x128 store icon with 16px transparent padding.
- `imagegen/promo-small-440x280.png` - required small promotional image.
- `imagegen/promo-marquee-1400x560.png` - optional marquee promotional image.
- `imagegen/screenshot-01-write-freely.png` - first screenshot; leads with "Write freely. Send confidently."
- `imagegen/screenshot-02-three-tones.png` - shows faithful, casual, and formal choices.
- `imagegen/screenshot-03-language-aware.png` - shows language-aware rewriting.
- `imagegen/screenshot-04-privacy.png` - shows privacy and sensitive-field exclusions.

## Captions

1. Write freely. Double-tap Shift when you are ready to send.
2. Send confidently with faithful, casual, or formal rewrites.
3. Write freely in one language and send confidently in another.
4. Draft freely with sensitive-field exclusions and no stored originals.

## Source Files

The imagegen originals are preserved in `imagegen/raw/`; the upload files in `imagegen/` have been resized/cropped to exact Chrome Web Store dimensions.
